temperature.py - программа. представляющая собой реализацию группового проекта №3 "Прогнозирование погоды по префиксным суммам"

temperature.cvs - данные, необходимые для коректной работы программы, файл содержит в себе информацию о температуре за каждый календарный день. Данные были сгенерированы
# -
Функционал программы

1. Программа выводит среднюю температуру за любой период - с помощью префиксных сумм
2. Вычисляет среднюю температуру за каждый календарный месяц - с помощью префиксных сумм
3. Находит самый тёплый и самый холодный день года - с помощью линейного поиска
4. Сортировка месяцев по по убыванию температуры - с помощью встроенной сортировки
5. Показывают температуру первых N дней - доступ по индексу
6. Сглаживает температурный ряд с помощью скользящего окна - с помощью очереди
7. Отмена последнего сглаживания - с помощью стека
8. Поиск дней в изначальных данных с температурой выше заданной - с помощью BTS
